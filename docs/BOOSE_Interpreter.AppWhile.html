Remarks

The while loop repeatedly executes a block of code as long as a specified condition evaluates to true. This implementation follows the standard while loop semantics where the condition is evaluated before each iteration of the loop body.

Methods

Compile()

Compiles the while loop command, setting up necessary control structures and program flow information.

public override void Compile()

Remarks

During compilation, this method:

  1. Calls the base class compilation logic
  2. Sets the conditional type to while loop
  3. Pushes this command onto the program's control structure stack
  4. Records the current line number for loop control

Execute()

Executes the while loop command by evaluating the condition and controlling program flow.

public override void Execute()

Remarks

The execution process:

  1. Evaluates the condition expression
  2. If the condition is an expression, evaluates it using the program's expression evaluator
  3. Converts the result to a boolean value
  4. If the condition is false, skips to the end of the loop block
  5. If the condition is true, continues to the next instruction (loop body)
